# SAR_ADC_STM32
Discrete 8-bit SAR ADC (STM32 Bare-Metal) While modern microcontrollers come with high-speed integrated ADCs, I wanted to strip away the "black-box" abstraction and implement the Successive Approximation Register logic from the ground up. This project is a discrete implementation of an 8-bit ADC using an STM32 and a custom R-2R resistor ladder.
